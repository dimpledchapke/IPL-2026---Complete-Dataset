# IPL-2026---Complete-Dataset
Data Analytic Project-IPL 2026 - Complete Dataset using Python, SQL, Tableau

🏏 Overview

This project focuses on analyzing the IPL 2026 dataset using Python, SQL, and Tableau to uncover insights related to team performance, player statistics, match trends, and venue analysis.

The project includes:

Data loading and preprocessing
Exploratory Data Analysis (EDA)
Data cleaning and transformation
SQL analysis on Databricks
Interactive Tableau dashboard creation
Report generation
Presentation creation using Gamma

The goal of this project is to transform raw cricket data into meaningful insights through analytics and visualization.

📂 Dataset

The dataset contains multiple CSV files related to IPL 2026 matches and player statistics.

Main Tables
matches.csv → Match-level information
deliveries.csv → Ball-by-ball data
batting_stats.csv → Batting performance
bowling_stats.csv → Bowling performance
fielding_stats.csv → Fielding statistics
points_table.csv → Team standings
venues.csv → Venue details
squads.csv → Team squad information

🔄 Project Steps

Match outcomes
